# 智能送货小车系统 — 测试报告

## 测试环境

| 项目 | 配置 |
|------|------|
| 小车 | Yahboom ROS2 小车 + Jetson Orin Nano |
| 激光雷达 | RPLIDAR A1 |
| 相机 | Astra Pro Plus 深度相机 |
| ROS2 | Humble (Docker 容器 a0e6) |
| 测试场地 | 逸夫楼 5 层走廊 |

## 功能测试

### Web 后台

| 用例 | 操作 | 预期 | 结果 |
|------|------|------|------|
| 创建订单 | 填写订单号、收件人、教室号 → 提交 | 订单创建，TCP 推送小车 | ✅ |
| 收件人录入 | 上传姓名+人脸照片 | 照片保存，数据库记录 | ✅ |
| 订单状态跟踪 | 小车配送过程 | 状态自动更新 | ✅ |
| 人脸自动匹配 | 选收件人后创建订单 | 自动调取人脸照片 | ✅ |

### 人脸识别

| 用例 | 操作 | 预期 | 结果 |
|------|------|------|------|
| 嵌入提取 | 录入 personA/B 照片 | 生成 enrolled_embeddings.npz | ✅ |

## 已知问题

| 问题 | 影响 | 状态 |
|------|------|------|
| 小车需在走廊（地图内）才能定位 | 必须从走廊起点出发 | 已知限制 |
| 导航超时后无自动恢复 | 需手动重试 | 后续优化 |
| 人脸核验依赖光照条件 | 弱光下可能失败 | 建议增加补光 |
