================================================================
  数据集存放说明
================================================================

将你的数据放到以下目录：

  TRAFFIC/
  ├── images/          ← 放所有图片 (*.jpg, *.png)
  │   ├── img001.jpg
  │   ├── img002.jpg
  │   └── ...
  ├── labels/          ← 放所有 YOLO 格式标注 (*.txt)
  │   ├── img001.txt   (每行: class_id x_center y_center width height)
  │   ├── img002.txt
  │   └── ...
  └── classes.txt      ← 类别列表 (已创建)

================================================================
  标注格式 (YOLO 归一化):
  每行: <class_id> <x_center> <y_center> <width> <height>
  所有坐标归一化到 [0, 1] 范围
  示例: 0 0.523 0.345 0.120 0.080

  class_id: 0=turnleft, 1=turnright, 2=turnaround
================================================================

放好数据后，运行:
  cd D:\CarApp\yolo_training
  python split_dataset.py

这会自动拆分数据为 train/valid/test 三个集合。
================================================================
